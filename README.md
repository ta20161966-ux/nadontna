# Training Planner

**Автор:** [Ваше Имя Фамилия]

**Описание:**
Приложение для планирования тренировок с возможностью добавления, фильтрации и сохранения данных в формате JSON.

**Примеры использования:**
1. Запустите приложение: python main.py
2. Введите дату, тип и длительность тренировки, нажмите «Добавить».
3. Используйте фильтрацию по типу или дате.
4. Данные сохраняются в файл data.json.
